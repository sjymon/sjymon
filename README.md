# Hi, I'm Otto 👋

**Software Engineer | Frontend & Mobile App Developer | Technical Lead**

I am a Software Engineer with 8+ years of experience in web and mobile application development, with strong expertise in Frontend development using **Angular**, **React**, **Next.js**, and **TypeScript**.

Since 2018, I have worked across multiple roles including Frontend Developer, Mobile App Developer, Backend Developer, DevOps support, and Technical Leader. I have over 3 years of experience leading technical solutions, mentoring team members, reviewing code, and improving delivery quality.

My main focus is building scalable, maintainable, and high-quality software products with clean architecture, strong code quality, and practical business value.
